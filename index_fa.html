مانیتور چند سنسوره IoT با ESP32

این پروژه یک کاربرد اینترنت اشیا (IoT) است که از یک میکروکنترلر ESP32 برای نظارت بر پارامترهای مختلف محیطی استفاده می‌کند. داده‌ها را از یک سنسور DHT21 (دما و رطوبت)، یک سنسور ضربان و یک سنسور گاز MQ135 (سطوح CO2) جمع‌آوری می‌کند. داده‌ها در یک صفحه وب میزبانی شده روی ESP32 نمایش داده می‌شود، جایی که کاربران همچنین می‌توانند یک لامپ متصل را روشن و خاموش کنند و داده‌های ثبت شده را دانلود کنند.

ویژگی‌ها

  • اتصال WiFi: به یک شبکه WiFi مشخص شده متصل می‌شود.
  • سنسورهای متعدد: داده‌ها را از سنسورهای DHT21، سنسور ضربان و سنسور MQ135 می‌خواند.
  • سرور وب: یک صفحه وب را برای نمایش داده‌های سنسور و کنترل یک لامپ میزبانی می‌کند.
  • SPIFFS: داده‌ها را به صورت محلی در سیستم فایل ESP32 ذخیره می‌کند.
  • JSON: از JSON برای تبادل داده بین ESP32 و رابط وب استفاده می‌کند.

سخت‌افزار مورد نیاز

  • میکروکنترلر ESP32
  • سنسور دما و رطوبت DHT21
  • سنسور ضربان
  • سنسور گاز MQ135
  • لامپ LED (اختیاری)
  • مقاومت‌ها، بردبورد و سیم‌های جامپر

نرم‌افزار مورد نیاز

  • Arduino IDE با پشتیبانی ESP32
  • کتابخانه ArduinoJson
  • کتابخانه سنسور DHT
  • کتابخانه WiFi
  • کتابخانه WebServer
  • کتابخانه SPIFFS
  • کتابخانه HTTPClient

نمودار مدار

سنسورها را به ESP32 به شرح زیر متصل کنید:

سنسور DHT21

  • VCC به 3.3V
  • GND به GND
  • DATA به GPIO4

سنسور ضربان

  • سیم بنفش به GPIO36

سنسور MQ135

  • خروجی آنالوگ به GPIO34

لامپ

  • پین کنترل به GPIO2 (با مقاومت مناسب)

توضیح کد

  1. شامل کردن کتابخانه‌ها: کتابخانه‌های لازم برای WiFi، سرور وب، سنسور DHT، SPIFFS و JSON را شامل می‌کند.
  2. پیکربندی WiFi: اعتبارنامه‌های WiFi را برای اتصال به شبکه تنظیم می‌کند.
  3. راه‌اندازی سنسورها: سنسورهای DHT21، سنسور ضربان و سنسور MQ135 را راه‌اندازی می‌کند.
  4. مسیرهای سرور وب: مسیرهایی را برای صفحه اصلی، روشن و خاموش کردن لامپ، خواندن داده‌های سنسور و دانلود داده‌ها تعریف می‌کند.
  5. صفحه اصلی: یک صفحه HTML را ارائه می‌دهد که داده‌های سنسور را نمایش می‌دهد و دکمه‌هایی برای روشن و خاموش کردن لامپ و دانلود داده‌ها فراهم می‌کند.
  6. مدیریت داده‌ها: داده‌های سنسور را می‌خواند، آن‌ها را در SPIFFS ذخیره می‌کند و به یک سرور از راه دور ارسال می‌کند.

راه‌اندازی و نصب

  1. نصب Arduino IDE: مطمئن شوید که Arduino IDE روی کامپیوتر شما نصب شده است.
  2. نصب برد ESP32 در Arduino IDE:
    • به File > Preferences بروید.
    • https://dl.espressif.com/dl/package_esp32_index.json را به Additional Board Manager URLs اضافه کنید.
    • Boards Manager را از Tools > Board باز کنید و بسته esp32 را نصب کنید.
  3. نصب کتابخانه‌های مورد نیاز:
    • به Sketch > Include Library > Manage Libraries بروید.
    • کتابخانه‌های زیر را نصب کنید: ArduinoJson، DHT sensor library، WiFi، WebServer، SPIFFS، HTTPClient.
  4. ESP32 را به کامپیوتر خود متصل کنید: از یک کابل USB برای اتصال ESP32 به کامپیوتر خود استفاده کنید.
  5. آپلود کد: کد ارائه شده را در Arduino IDE باز کنید و آن را به ESP32 آپلود کنید.

اجرای پروژه

  1. روشن کردن ESP32: پس از آپلود کد، ESP32 را با استفاده از اتصال USB یا یک منبع تغذیه خارجی روشن کنید.
  2. اتصال به WiFi: ESP32 تلاش خواهد کرد به شبکه WiFi مشخص شده متصل شود.
  3. دسترسی به رابط وب:
    • یک مرورگر وب را در دستگاهی که به همان شبکه WiFi متصل است باز کنید.
    • آدرس IP مربوط به ESP32 که در Serial Monitor نمایش داده شده است را وارد کنید.
  4. نظارت بر داده‌ها: داده‌های دما، رطوبت، سطوح CO2 و ضربان قلب را مشاهده کنید، لامپ را روشن و خاموش کنید و داده‌های ثبت شده را از رابط وب دانلود کنید.
