Multi-Sensor IoT Monitor with ESP32

Introduction

This project is an IoT solution for monitoring environmental conditions using the ESP32 microcontroller. The data is collected from various sensors and presented through a web server with a PHP backend and an HTML frontend.

Features

  • Real-time environmental monitoring
  • Data visualization through a web interface
  • Integration with multiple sensors

Installation

To get started, clone this repository and follow the installation instructions in the GitHub repository.

Contribution
